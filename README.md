# bcjoy.py

🐼 一键加入你喜欢的 BearyChat 团队

## 功能

✔️️ 团队简介 landing page
✔️️ 加入团队 svg badge

## 安装 & 使用

### self-host

```
$ git clone https://github.com/bearyinnovative/bcjoy.py.git
```

如果你用 [pipenv][]:

```
$ pipenv install
```

或者 pip:

```
$ pip install -r requirements.txt
```

[pipenv]: https://github.com/kennethreitz/pipenv
